# 📸 Code Screenshot Manager

A simple and practical web application built for personal use to organize coding screenshots and snippets efficiently.

## 🚀 Why I Built This

While learning and working on projects, I often saved useful code snippets as screenshots. Over time, it became difficult to find them again.

So I built this tool to:

* Keep everything organized
* Quickly search important snippets
* Avoid wasting time searching old files

## ✨ Features

* 📁 Create and manage folders
* 🖼️ Upload and store code screenshots
* 🔍 Search by title and tags
* ⚡ Fast and simple UI
* 🧩 Tag-based organization
* 🗑️ Delete snippets

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, Bootstrap, JavaScript, jQuery
* **Backend:** PHP (PDO)
* **Database:** MySQL
```

## ⚙️ How to Run

1. Clone the repository
2. Move project to `htdocs` (XAMPP)
3. Create database and import tables
4. Update `db.php` with your database credentials
5. Run project on:

```
http://localhost/your-project-folder
```
